## Что понадобится перед началом

1. **Операционная система**: Windows 10 или Windows 11
2. **Установленный .NET 8.0** - скачать можно с официального сайта:
   https://dotnet.microsoft.com/download/dotnet/8.0
   Выберите версию ".NET 8.0 SDK" для Windows

## Пошаговая инструкция

### Шаг 1: Проверка установки .NET

1. Откройте командную строку (нажмите Win+R, введите `cmd` и нажмите Enter)
2. Введите команду: `dotnet --version`
3. Если вы видите версию 8.0.ххх - все установлено правильно
4. Если выводится ошибка - переустановите .NET 8.0 SDK

### Шаг 2: Скачивание исходного кода

1. Перейдите на страницу проекта на GitHub
2. Нажмите зеленую кнопку "Code"
3. Выберите "Download ZIP"
4. Распакуйте скачанный архив в удобную папку (например, `C:\KG-Zapret`)

### Шаг 3: Сборка проекта

1. Откройте командную строку (как в шаге 1)
2. Перейдите в папку с проектом. Например, если вы распаковали в `C:\KG-Zapret`, введите:
   ```
   cd C:\KG-Zapret
   ```
3. Выполните команду сборки:
   ```
   dotnet build
   ```

### Шаг 4: Проверка сборки

1. После успешной сборки в папке `bin\Debug\net8.0-windows` появится файл `KG-Zapret.exe`
2. Запустите его двойным щелчком мыши
3. Программа должна запуститься

## Возможные проблемы и решения

### Проблема: "dotnet не является внутренней или внешней командой"
**Решение**: Переустановите .NET 8.0 SDK и перезагрузите компьютер

### Проблема: Ошибки при сборке
**Решение**:
1. Убедитесь, что вы в правильной папке с файлом `KG-Zapret.csproj`
2. Попробуйте выполнить команду: `dotnet clean` а затем снова `dotnet build`

### Проблема: Программа запускается, но ничего не происходит
**Решение**: Запустите программу от имени администратора (правой кнопкой мыши -> "Запуск от имени администратора")

## Создание установочного пакета

Для создания единого исполняемого файла:

1. В командной строке выполните:
   ```
   dotnet publish -c Release -r win-x64 --self-contained true
   ```
2. Готовый файл будет в папке `bin\Release\net8.0-windows\win-x64\publish\`

## Получение помощи

Если у вас возникли проблемы со сборкой:
1. Проверьте, что все шаги выполнены точно
2. Убедитесь, что .NET 8.0 установлен корректно
3. Если проблема остается - создайте issue на GitHub с описанием ошибки

Примечание: Для работы программы требуются права администратора, так как она изменяет системные настройки.
